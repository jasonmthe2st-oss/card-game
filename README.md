# C++ Memory Game!

This is a quick little game made in c++ which is a quick memory matching game using CLI-representations of a 52-card deck.

## Card Display System

I created a little display system for cards that I'm very proud of! The face on the card corresponds to the value of the card; the eyes correspond to the rank of the card and the mouth corresponds to the suit. I will likely use this system for future projects.

## Compilation
This game can be compiled using normal means; memorygame.cpp is the only file which needs to be compiled.
However, note that the game uses Unicode characters to draw cards to the terminal. 
In my testing, these did not compile correctly on Windows, so be careful. Play with your compiler settings.
